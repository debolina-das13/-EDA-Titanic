# -EDA-Titanic

## EDA Steps
- Data overview using .info(), .describe(), and .value_counts()
- Univariate, bivariate, and multivariate analysis
- Visualizations using histograms, boxplots, pairplots, and heatmaps
- Handling missing values
- Summary of insights

## Key Findings
- Females had a higher survival rate
- First class passengers survived more
- Fare positively correlated with survival
- Dataset contained outliers and skewness
